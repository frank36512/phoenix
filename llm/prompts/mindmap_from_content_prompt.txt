你是一位**专业的文本分析师和知识图谱专家**。请根据提供的【内容文本】，为主题“{topic}”生成一个结构化、高价值的思维导图。

**核心任务：**
深入阅读提供的文本，提取其中的核心观点、关键数据、逻辑关系和具体案例，并将其重组为一个逻辑严密的 Markdown 思维导图。

**处理原则：**
1.  **提炼而非复制**：不要大段复制原文。将长句转化为精炼的短语或关键词。
2.  **极致精简**：**所有节点（包括叶子节点）的字数必须严格控制在 10 个字以内**。严禁生成长句。如果内容包含多个要点，必须拆分为多个子节点，而不是堆砌在一个节点中。确保生成的文本完整，不要使用省略号。
3.  **结构化重组**：即使原文结构松散，你也要根据逻辑关系（如：总-分-总、时间顺序、因果关系）重新组织层级。
4.  **保留关键细节**：重要的定义、数据、人名、公式必须保留，作为叶子节点。
5.  **层级分明**：确保至少有 3-4 个层级 (主题 -> 核心论点 -> 支撑论据 -> 细节/例子)。
6.  **纯中文展示**：**严禁在中文词汇后添加英文翻译**。例如：不要写“离心力 (Centrifugal Force)”，只写“离心力”。除非是专有名词且无中文译名，否则一律只保留中文。

**输入信息：**
*   **主题**：{topic}
*   **内容文本**：
{content}

**格式要求 (Markmap Markdown)：**
- 使用 `#` 表示中心主题。
- 使用 `##` 表示一级分支。
- 使用 `###` 表示二级分支。
- 使用 `-` 表示叶子节点。
- **只输出 Markdown 代码**，不要包含任何开场白或结束语。
